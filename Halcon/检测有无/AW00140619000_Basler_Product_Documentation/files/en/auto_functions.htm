Auto Functions

Auto functions allow you to control image properties automatically.

Auto functions are particularly useful to maintain good image quality when imaging conditions change frequently, e.g., unreliable light conditions.

Most auto functions are the automatic counterparts to setting a parameter manually. For example, the Gain Auto feature controls the GainRawGain parameter automatically within specified limits.

Available Auto Functions

Depending on your camera model, the following auto functions are available:

  • Gain Auto
  • Exposure Auto
  • Balance White Auto
  • Pattern Removal Auto
  • Tonal Range Auto
  • Gain/Exposure Auto

Operation

The individual auto functions can be used at the same time.

If you are using Exposure Auto and Gain Auto at the same time, you can use the Auto Function Profile feature to specify how the effects of gain and exposure time are balanced.

The pixel data for the auto functions can come from one or multiple auto function ROIs. To operate properly, at least one auto function ROI must be assigned to each auto function.
