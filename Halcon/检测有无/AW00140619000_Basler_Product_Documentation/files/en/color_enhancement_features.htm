Color Enhancement Features

Color enhancement features allow you to adjust the appearance of the colors in your images.

Available Color Enhancement Features

Depending on your camera model, the following color enhancement features are available:

  • Balance White
  • Balance White Adjustment Damping
  • Balance White Auto
  • Balance White Reset
  • Color Adjustment
  • Color Transformation
  • Hue and Saturation
  • Light Source Preset
  • Processed Raw Enable
  • Tonal Range
  • Tonal Range Auto
