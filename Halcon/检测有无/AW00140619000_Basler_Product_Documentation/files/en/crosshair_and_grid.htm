Crosshair and Grid

Use the crosshair and grid as visual aids when positioning your camera to get a perfect shot of your target object.

Using Crosshair and Grid

To show/hide the crosshair or grid:

  1. Start image acquisition by clicking the single or continuous shot button in the toolbar.
  2. Click the or buttons in the toolbar.
    Alternatively, you can right-click in the image display area and select Show Crosshair or Show Grid in the context menu. The context menu also offers options to move the crosshair to a different position in the image.

The color of the crosshair adapts automatically to the color in the image. This ensures that it is always clearly visible.

To move the crosshair:

  1. Right-click in the image display area and select Move Crosshair.
    Alternatively, you can set the position by providing X and Y offset values in the View Options dialog.
  2. When the crosshair is at the desired position, right-click again and select Lock Crosshair.

When moving the crosshair, displaying the grid helps you to find the desired position.

To move the crosshair back to its original position in the center of the image, select Center Crosshair in the context menu.

Modifying Crosshair and Grid

You can modify the appearance of the crosshair and grid. To do so, select View Options in the context menu or in the View menu.

On the Crosshair page of the View Options dialog, you can change the line width of the crosshair. The X Offset and Y Offset options offer an alternative method for moving the crosshair. The starting point for any move operation is the center of the image. Therefore, moving the crosshair to the left or up will result in negative values. Click the Center button to move the crosshair back to its original position.

On the Grid page of the View Options dialog, you can set the desired grid size (fine, default, or coarse).
