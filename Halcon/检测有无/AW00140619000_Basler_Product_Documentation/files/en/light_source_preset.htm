Light Source Preset

The Light Source Preset camera feature allows you to correct color shifts caused by certain light sources.

Using the Feature

Prerequisites

The pixel format must be set to one of the supported pixel formats.

Selecting a Light Source Preset

To select a light source preset, set the LightSourceSelectorLightSourcePreset parameter to one of the following values:

  • OffOff: No light source preset is selected.
  • DaylightDaylight5000K: The camera corrects color shifts caused by daylight lighting that has a color temperature of about 5 000 K.
  • Daylight6500KDaylight6500K: The camera corrects color shifts caused by daylight lighting that has a color temperature of about 6 500 K.
  • TungstenTungsten2800K: The camera corrects color shifts caused by tungsten lighting that has a color temperature of about 2 500 to 3 000 K.
  • MicroscopeLED4500K: If available, the camera corrects color shifts caused by microscope LED lighting that has a color temperature of about 4 500 K.
  • MicroscopeLED5500K: If available, the camera corrects color shifts caused by microscope LED lighting that has a color temperature of about 5 500 K.
  • MicroscopeLED6000K: If available, the camera corrects color shifts caused by microscope LED lighting that has a color temperature of about 6 000 K.
  • Fluorescent4000K: If available, the camera corrects color shifts caused by fluorescent lighting that has a color temperature of about 4 000 K.
  • Custom: If available, selecting this preset enables the Color Transformation feature which allows you to customize the light source settings. You should only select this preset if you are thoroughly familiar with matrix color transformations. The camera also adjusts the Balance White and Color Adjustment settings so that they have neutral values that do not change the appearance of the colors.

The default light source preset varies by camera model.

How It Works

Depending on its specific color temperature, the light used for image acquisition can cause color shifts in the image. You can correct these color shifts by selecting the related light source preset.

When you select a light source preset, the camera automatically adjusts the settings of the following color enhancement features:

  • Balance White
  • Color Adjustment
  • Color Transformation

The settings will be optimized for the selected light source.

// Disable light source presets (no correction)

// Set the light source preset for daylight (at about 5000K)
camera.LightSourcePreset.SetValue(LightSourcePreset_Daylight5000K);
// Set the light source preset for tungsten lighting
