Features

Features are capabilities of Basler cameras and modules that can be controlled by configuring firmware parameters.

You can configure the parameters via the Basler pylon API or via the Basler pylon Viewer.

Browse Features

  1. In the Basler Product Documentation toolbar, click the  icon.
    The Table of Contents pane opens.
  2. Navigate to the Cameras > Features section.
    The color of the entries indicates whether a feature is available:

(1) Feature is supported by the camera model currently selected.

(2) Feature is not supported by the camera model currently selected.

Search Features

  1. In the Basler Product Documentation toolbar, click the icon.
    The Search pane opens.
  2. Type the name of the feature, e.g., "binning", and press Enter.
  3. In the list of search results, click the desired entry.
    The topic is displayed.
    If the feature is not supported by the camera model currently selected, a message box is displayed that allows you to change the camera model.
