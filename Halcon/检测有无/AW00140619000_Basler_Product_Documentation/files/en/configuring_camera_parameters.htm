Configuring Camera Parameters

The pylon Viewer application allows you to configure most of the camera parameters using a graphical user interface.

The following animation demonstrates how to configure camera parameters with the pylon Viewer.

As an example, the Gain Auto auto function is set to Off and the Gain and Black Level parameter values are increased.

To quickly access the parameters, the Search feature is used.

If the parameter settings that you want to adjust don't appear in the feature tree, choose a different user level.
